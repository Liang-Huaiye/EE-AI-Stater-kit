# EE-AI-Stater-kit
# 🚀 EE-AI-Starter-Kit | 电子信息工程 AI 提效指南

这是专为 EE（电子信息工程）学生打造的 AI 提效工具库。旨在利用 AI（Claude/GPT/DeepSeek）解决硬件编程、信号处理、实验报告中的枯燥环节。

## 🌟 核心模块 (Roadmap)
- [ ] **AI + Verilog/FPGA**: 自动生成 Testbench 与状态机调试。
- [ ] **AI + 嵌入式 (STM32/ESP32)**: 快速配置寄存器与 HAL 库代码。
- [ ] **AI + 信号处理**: 用 Python 快速实现滤波器设计与可视化。
- [ ] **AI + 论文/实验报告**: LaTeX 公式转换与学术润色。

## 🛠 当前可用工具：Verilog 状态机生成器 Prompt
> **使用方法**：将以下 Prompt 复制给 AI，描述你的需求即可。

### [Prompt 模板]
"你现在是一名资深 FPGA 工程师。请帮我用 Verilog 写一个状态机。
需求：[在此输入你的逻辑，如：三层电梯控制/自动售货机]
要求：符合 FSM 三段式写法，带有详尽的中文注释，并提供一个 Testbench。"
